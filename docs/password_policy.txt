PASSWORD SECURITY POLICY
Version: 5.1
Department: IT Security
Classification: Internal

Requirements:
- Minimum password length: 14 characters
- MFA required for privileged accounts
- Password reuse prohibited
- Shared accounts prohibited

Approved Password Managers:
- Bitwarden
- 1Password
- KeePassXC

Credential Exposure Response:
1. Reset affected credentials.
2. Invalidate active sessions.
3. Review authentication logs.
4. Investigate suspicious activity.
